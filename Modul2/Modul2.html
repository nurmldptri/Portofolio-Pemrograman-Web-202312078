Praktikum CSS

CSS adalah singkatan dari Cascading Style Sheets, yaitu bahasa pemrograman yang digunakan untuk mengatur tampilan dan format dari elemen-elemen HTML di sebuah halaman web. CSS memungkinkan kita untuk mengatur warna, ukuran, font, layout, dan animasi pada elemen HTML. Berikut merupakan Jenis CSS:

Inline CSS
Internal CSS
External CSS
